Ddifference bet spring frame work & Spring boot

    Spring FrameWork                                        Spring Boot
    1) requires manual config                              Auto Config
    2) More Setup                                          less
    3) no embedded server                                  embedded server
    4) Slower Development                                  faster

        @Service : Service Layer (Model MvC)
        @Repository - DataBase Layer
        @SpringBootApplication - main spring boot application
